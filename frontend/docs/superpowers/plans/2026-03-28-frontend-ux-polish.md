# Frontend UX/UI Polish Plan (2026-03-28)

## 1. RAG 智能问答区体验升级 (`FinancialQA.vue` & `ChatHistory.vue`)
- **对话气泡打磨**：区分用户和 AI 的视觉层级（不同背景色、微阴影、头像占位符），增加文本排版的行高和字间距。
- **“思考中”状态动画**：在等待后端 RAG 接口返回时，将简单的 `Loading...` 文本替换为骨架屏或呼吸点动画（如 `...` 跳动）。
- **高级引用 (Citations) 交互**：当 AI 回复带有文档来源时，将纯文本来源升级为精美的小 Tag 或脚注卡片；鼠标悬浮 (Hover) 时显示原文片段 (Snippet)。
- **丝滑滚动**：每次发问或收到长回复后，确保聊天窗口能平滑自动滚动到底部。

## 2. 知识库管理区增强 (`KnowledgeBase.vue`)
- **文件上传进阶**：从基础的 `<input type="file">` 升级为支持拖拽 (Drag & Drop) 的上传虚线框区域，并配以明确的上传图标。
- **状态徽章可视化**：将文档处理状态映射为带颜色的 Badge：
  - `uploaded` / `parsed` / `chunked`: 蓝色或黄色提示（处理中）
  - `indexed`: 绿色（成功）
  - `failed`: 红色（失败）
- **空状态 (Empty States)**：在列表为空时显示优雅的占位提示（如“暂无文档，快去上传”）。

## 3. 全局框架与反馈层 (`Workbench.vue` & `AdminUsers.vue`)
- **全局轻提示 (Toast)**：实现或引入一个简单的 Toast 组件，替换掉难看的 `alert()` 弹窗（用于上传成功、报错等）。
- **表格与角色高亮**：管理后台的用户列表增加鼠标悬停高亮；不同权限组 (`super_admin`, `admin`, `member`) 配以不同底色的标签。
